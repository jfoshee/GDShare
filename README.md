# Integrated GDShare

This is a mod for GD that adds buttons to export / import levels.

## Installation

The mod can be installed through the in-game Downloads page in [Geode](https://geode-sdk.org). Search for "GDShare" to find it.

## GMD File Format

For detailed information on the GMD file format, including how to read and write GMD files, please refer to the [GMD_Format.md](GMD_Format.md) file.
